## 關鍵自動化流程

### 🪵 木材生產線

> 🔑 關鍵寶可夢：雙斧戰龍（亂灑 + 伐木）

```mermaid
flowchart LR
    HX["雙斧戰龍"] -->|"亂灑"| LOG["小圓木"]
    LOG -->|"分類"| B1["📦 大家的盒子"]
    EXTRA["📥 額外木材輸入"] -.-> B1
    B1 -->|"伐木（雙斧戰龍）"| WOOD["🪵 木材"]
```

### 🌾 農場

```mermaid
flowchart LR
    FARM["農場"] -->|"栽培"| GROW["加速成長"]
    SCAT["亂灑素材"] -->|"分類"| BOX["大家的盒子"]
```

### ⛏️ 礦物自動化

> 🔑 關鍵寶可夢：小炭仔一家（分類 + 點火）

```mermaid
flowchart LR
    WP["烏波 / 土王"] -->|"亂灑"| CLAY["軟塌塌黏土"]
    GL["晶光芽 / 晶光花"] -->|"亂灑"| IRON["鐵"]
    GB["臭泥 / 臭臭泥 / 灰塵山"] -->|"亂灑"| TRASH["不可燃垃圾"]

    CLAY -->|"分類"| BOX["📦 大家的盒子"]
    IRON -->|"分類"| BOX
    TRASH -->|"分類"| BOX
    BOX -->|"回收利用🔑灰塵山"| IRON_R["鐵"]
    IRON_R --> BOX
    EXTRA["📥 額外礦物輸入"] -.-> BOX

    BOX -->|"點火🔑小炭仔一家"| BRICK["🧱 磚塊"]
    BOX -->|"點火🔑小炭仔一家"| IBLOCK["⚙️ 鐵塊"]
```

> 🔑 垃圾線關鍵：灰塵山（亂灑 + 回收利用）

### 🔨 碾壓生產線

> 岩石系寶可夢的碾壓專長，可將樹果與石灰加工

```mermaid
flowchart LR
    subgraph dye["🎨顏料"]        
        direction LR
        BERRY["樹果"] -->|"碾壓"| DYE["顏料"]
    end

    subgraph cement["🏗️水泥"]
        direction LR
        LIME["石灰"] -->|"碾壓"| CEMENT["水泥"]
    end
```
